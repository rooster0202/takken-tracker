# 宅建士 問題チェック

章ごとの問題を「○・△・×」で自己評価する個人用Webアプリです。

## 使い方

`index.html` をブラウザで開くと問題チェックに使えます。評価はブラウザに自動保存されます。

`dashboard.html` では、保存済みの評価をもとに分析を確認できます。

- `○`: できた
- `△`: 迷った、復習したい
- `×`: できなかった

各問には復習用のメモを保存できます。

各問には分析用の分類タグも表示しています。

分析ページで全体と章ごとの評価状況、○△×の内訳、復習候補数、分類別の復習候補、自動分析を確認できます。

収録数は、Chapter1 109問、Chapter2 100問、Chapter3 63問、Chapter4 46問です。

iPadのSafariで共有メニューから「ホーム画面に追加」すると、宅建チェック用のアイコンで開けます。
